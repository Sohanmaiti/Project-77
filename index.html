Visual Studio Code


What is Visual Studio Code ?

Visual Studio Code is a code editor made by Microsoft for Windows, Linux and macOS. Features include support for debugging, syntax highlighting, intelligent code completion, snippets, code refactoring, and embedded Git. Users can change the theme, keyboard shortcuts, preferences, and install extensions that add additional functionality.


How to set it up ?

1. Download and install VS Code.

2. Create a new file.

3. See an overview of the user interface.

4. Install support for your favorite programming language.

5. Change your keyboard shortcuts and easily migrate from other editors using keymap extensions.

6. Customize your editor with themes.

7. Explore VS Code features in the Interactive Editor Playground.

Web Console


What's Console ?

Console can be said to be a list of functions which are hidden from the user and these are used in carrying out various activities in the webpage. These functions can be seen on clicking on Inspect option by right-clicking on the website.

What's it's function ?

The main function of the console is to Log the information associated with a web page.


How to print Data in the console screen and how to open it ?

In order to print data in the console screen one has to first create the initial part like the head and body under the html tag. Once it's done inside the body open a script tag and then put the different functions in different variables. Then attach the variables to the console by using console.log(var_name). Don't forget to give semi-colons (;) after each line.
In order to view the console data right-click on the web-page and select Inspect. A dialog-box will appear, ther from the top navigator select console. You will be able to see a list of data.
